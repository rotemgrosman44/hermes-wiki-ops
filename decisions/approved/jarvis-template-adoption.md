# Jarvis Template Adoption

## Decision
Hermes may adopt structure from selected Jarvis templates, but not Jarvis canon content.

## Approved Copy-Safe Imports
1. Telegram deliverables shape.
2. Readable status contract shape.
3. Opportunity card schema shape.
4. Execution boundary policy shape.
5. Generic naming and wikilink style.

## Not Approved
- Canon pages.
- Jarvis memory pages.
- Live authority notes.
- Historical canon content.
- Any page whose value depends on being the official source of truth.

## Reason
Hermes needs stable structure, not canon duplication.
