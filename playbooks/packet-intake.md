# Packet Intake

## Standard Flow
1. Read README and instruction files first.
2. Validate required files exist.
3. Validate checksums before analysis.
4. Store immutable source material under raw/ if worth keeping.
5. Update or create a bounded project or concept page.
6. Append a log entry.

## Output Rule
Only validated, bounded outputs move back through the bridge.
