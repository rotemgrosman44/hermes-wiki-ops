# Wiki Usage Policy

## When Hermes Writes
- After validating a packet.
- When a playbook becomes reusable.
- When a project note has durable value.
- When a decision needs a bounded record.

## When Hermes Does Not Write
- Passing mentions.
- Unvalidated rumors.
- Temporary chatter.
- Anything that belongs only in Jarvis canon.

## Required Loop
1. Validate.
2. File raw material if worth keeping.
3. Update a durable page.
4. Update index if a new durable page was created.
5. Append log entry.
